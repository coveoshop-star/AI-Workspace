# 原始输入

这是一条用于测试 AI Knowledge OS 流程的手动资料。

测试目标：

1. 验证新资料可以进入 `00-Inbox`。
2. 验证 AI 可以把 Inbox 内容整理成 `08-Pending` 审核稿。
3. 验证 Pending 审核稿可以指向正式项目文件。

# 背景

当前 ChewyWell 的本周重点是 SEO 体检、首页优化、产品页 FAQ、Prompt 库和 SOP 建立。

首页 SEO 需要优先检查：

- Title
- Meta Description
- H1
- Hero 文案
- Trust Badges
- Newsletter 入口

# 期望输出

生成一条 Pending 审核稿，说明：

- 这条资料和哪个项目相关
- 应该写入哪些正式文件
- 下一步应该怎么做
- 哪些内容需要人工确认
